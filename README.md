# B.J. Blazkomeat
A Wolfenstein 3D inspired character for H3VR Take and Hold.
![Thumbnail](/BJBlazkomeat/BJBlazkomeat/thumb.png)

## Requirements
* [Take And Hold Tweaker](https://github.com/devyndamonster/TakeAndHoldTweaker)
* [Deli Mod Loader](https://github.com/Deli-Counter/Deli)
* [BepInEx](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.5)

## Installation
Grab the latest zip file from the Releases page and drop it into the `/mods` folder in your H3VR directory.

## Details
In this character you have a limited arsenal available based on the original Wolfenstein 3D game from 1992, including:
* P38 / LugerP08 

![Pistol](/BJBlazkomeat/BJBlazkomeat/pistol.png)


* MP40 

![SMG](/BJBlazkomeat/BJBlazkomeat/smg.png)


* M134Minigun (Chaingun) 

![Chaingun](/BJBlazkomeat/BJBlazkomeat/chain.png)


* PanzerSchreck54 (Launcher) 

![Launcher](/BJBlazkomeat/BJBlazkomeat/launcher.png)


* Stielhandgranate 

![Grenade](/BJBlazkomeat/BJBlazkomeat/grenade.png)

## Enjoy :)
I recommend putting your favourite Wolfenstein 3D tracks on YouTube via SteamVR desktop and playing on the default map, due to its narrow corridors somewhat similar to Wolfenstein 3D.
